Class: ThrowableObject

ThrowableObject(x, y, speed)

Represents a throwable object (such as a grenade) in the game world that can be thrown, animated with rotation, and eventually trigger an explosion. Inherits from MovableObject.

Constructor

new ThrowableObject(x, y, speed)

Creates a new ThrowableObject instance. Loads the initial grenade image, explosion and rotation animations, sets the object's position and speed, applies gravity, and starts the throw action.
Parameters:
Name Type Description
x number The initial x-coordinate of the throwable object.
y number The initial y-coordinate of the throwable object.
speed number The horizontal speed at which the object moves.
